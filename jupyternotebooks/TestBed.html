C++ programs rely upon libraries – you will not get far without at least one of these, because they include basic ‘facilities’ needed such as the underpinnings of how to actually write to a terminal window etc…

c++ #include </code: c++> is an instruction that the contents of the header files is copied into the cpp file at the point of inclusion when the build occurs.

A file included by #include is often called a header file, which come in two types. Standard Library Header files have their names in angle brackets, with out a suffix e.g. . User-define Header files the ending .h, or .hpp, and are placed in speech marks, such as #include “myHeader.h” , are used for lists of values, functions to be reused, class and structures etc…
